# Сайт-визитка услуг (ветка feature/services-landing)

Эта ветка предназначена для адаптации текущего SPA под сайт-визитку без каталога товаров:

- Главный фокус: разделы услуг, портфолио, контакты, CTA (звонок, WhatsApp/Telegram).
- Отключаем: корзину, каталог товаров, сравнение, избранное.
- Навигация: упрощённая — хеш-маршруты на #services, #portfolio, #contacts.
- PWA/манифест: сохраняем, но минимизируем пред-кеш до ключевых статичных ассетов.

План предстоящих изменений:
1. Конфиг навигации: оставить только секции услуг/портфолио/контакты.
2. Удалить вызовы рендера каталога и карточек товаров; скрыть соответствующие контейнеры.
3. Упростить хедер: убрать элементы каталога/корзины, оставить CTA.
4. Убрать cart.js зависимости из main.js (флаги и кнопки);
5. Проверить i18n ключи и удалить лишние (товары, корзина).
6. Обновить сервис-воркер: список кеша и стратегия.

После согласования — начну по чеклисту, держа совместимость с архитектурой проекта.
